# Instructions

## Goal
Train and validate a surrogate model for an FSI dataset.

## What to do

1. **Prepare the dataset**
   - Download the file `dataset.pickle` from this folder  
   - Upload it to your Google Drive (e.g. in your main folder or a dedicated course folder)  

2. **Open the notebook in Colab**  
   - [Open in Colab](https://colab.research.google.com/github/lucalomazziPolimi/Pia-Enhance-Course/blob/main/04_workshops/01_workshop_fsi/FSI_DL_session.ipynb)

3. **Run the setup cells**
   - Mount your Google Drive  
   - Make sure the dataset path in the notebook points to the location of `dataset.pickle`  

4. **Run the notebook step by step**
   - Preprocess the dataset  
   - Train the surrogate model  
   - Evaluate the results  

## Expected outcome

You should obtain:
- A trained surrogate model  
- Basic validation plots showing model performance  

## Notes

- Make sure the dataset path is correct, otherwise the notebook will not run  
- If you encounter errors, restart the runtime and run all cells again
