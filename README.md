# Deep Learning–Driven Design under Dynamic Loads

Course materials for *Deep Learning–Driven Design under Dynamic Loads*.

This repository contains lecture slides, notes, Colab notebooks, workshop material, and challenge resources for the course.

## How to use this repository

You do **not** need Git.

- To access slides and notes, open the relevant folder and download the PDF files.
- To run the coding material, click the **Open in Colab** links provided.
- Before each session, reopen notebooks from this repository in case there were updates.

## Repository structure

### 0. Introduction

### 2. Deep Learning for Engineering
- DL foundations for regression and surrogate modeling
- Inverse design

### 3. Metamaterials

### 4. Workshops

- Generative meta-design

### 5. Challenge
- Team-based design and simulation challenge

### 6. Experiments
- Experimental validation and testing
