# Deep Learning–Driven Design under Dynamic Loads

**Institution:** Politecnico di Milano / NTNU

**Format:** Intensive short course  
**Instructors:**  
- Dr. Lomazzi (PoliMi)  
- Prof. Aune (NTNU)

## Course Description

This course introduces deep learning approaches for the design of lightweight structures subjected to dynamic loading conditions.

It combines physics-based modeling (structural dynamics and fluid–structure interaction) with data-driven methods, focusing on surrogate modeling, topology optimization, and generative design.

The course is highly application-oriented and includes hands-on workshops, a team-based design challenge, and experimental validation.

## Learning Objectives

By the end of the course, students will be able to:

- Understand the effects of dynamic loading and fluid–structure interaction on structural behavior  
- Formulate engineering problems suitable for deep learning approaches  
- Develop surrogate models for complex simulations  
- Apply deep learning to inverse design tasks  
- Implement generative design workflows for lattice metamaterials  
- Critically evaluate model performance and limitations  
- Apply data-driven design methods in a practical engineering context

## Prerequisites

Students are expected to have:

- Basic knowledge of structural mechanics  
- Familiarity with linear algebra and calculus  
- Basic programming skills

No prior experience with deep learning and numerical simulations is required, but it is recommended.

## Course Format

The course follows a blended and hands-on format:

- Lectures covering theoretical foundations  
- Coding sessions using Google Colab  
- Guided workshops on real engineering problems  
- A team-based challenge focused on design under dynamic loading  
- Experimental validation of selected designs  

All coding activities are performed in Google Colab. No local installation is required.
